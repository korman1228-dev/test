# 1v1 Tetris (Browser Game)

**1v1 Tetris**는 웹 브라우저에서 즐길 수 있는 로컬 1대1 테트리스 게임입니다.  
왼쪽 플레이어와 오른쪽 플레이어가 서로 경쟁하며 점수를 올리는 PvP 게임입니다.  

블록 색상과 UI를 통해 플레이어를 구분할 수 있고, 줄 삭제 시 화면이 진동하며 승패를 확인할 수 있습니다.

---

## 🎮 특징

- 로컬 1대1 플레이 (한 화면)
- 각 플레이어 색상 구분: **P1 = 파랑**, **P2 = 빨강**
- 좌우 컨트롤 반전 옵션 적용
- 줄 삭제 시 화면 진동 효과
- 점수, 최고 점수 표시
- 게임 오버 시 승패 표시
- 다음 블록 표시

---

## 🕹 조작법

| 플레이어 | 이동 | 회전 | 하드 드롭 |
|----------|------|------|-----------|
| P1       | ← → (좌우 반전) | ↑ | `/` |
| P2       | A D (좌우 반전) | W | Shift |

- **R 키**: 게임 재시작 (게임 오버 후)
